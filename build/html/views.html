recipes.views.create_recipe(request)¶

Enables users to create a new recipe.

Supports both GET (displaying the form) and POST (saving the recipe).

recipes.views.home(request)¶

Displays the home page of the application.

recipes.views.search_recipes(request)¶

Allows users to search for recipes by name or ingredients. Also provides charts for visualizing recipe data.

recipes.views.signup_view(request)¶

Allows new users to sign up to the application.

Supports both GET (displaying the signup form) and POST (registering the user).
